# Twitter Sentiment Analysis Project

This project uses **Natural Language Processing (NLP)** and Machine Learning to classify tweets as either **Positive** or **Negative**. 

## 📊 Model Performance
- **Accuracy:** 77.6% 
- Developed using **Google Colab** and Python's Scikit-Learn library.

## 🛠️ Key NLP Techniques
- **Data Preprocessing:** Cleaned tweet text by removing stopwords, URLs, and special characters.
- **Stemming/Lemmatization:** Reduced words to their root form for better analysis.
- **Vectorization:** Converted text data into numerical format (using TF-IDF or Bag of Words).
- **Model Training:** Trained a classification algorithm (like Logistic Regression or SVM) to predict sentiment.

## 📁 Repository Contents
- `Twitter_Sentiment_Analysis.ipynb`: Full Google Colab notebook with code and visualizations.
- `trained_model.sav`: The saved, ready-to-use model file.
